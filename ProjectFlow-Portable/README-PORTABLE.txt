========================================
  ProjectFlow - Portable Version
========================================

🚀 QUICK START:

1. Make sure Node.js is installed on your computer
   Download from: https://nodejs.org

2. Install the 'serve' package (one-time setup):
   Open terminal/command prompt and run:
   npm install -g serve

3. Start ProjectFlow:
   - Windows: Double-click "start-projectflow.bat"
   - Mac/Linux: Run "./start-projectflow.sh" in terminal

4. Open your browser to: http://localhost:3000

========================================

📁 WHAT'S INCLUDED:

- out/ folder: The complete ProjectFlow application
- start-projectflow.bat: Windows startup script
- start-projectflow.sh: Mac/Linux startup script
- This README file

========================================

🔧 MANUAL START (if scripts don't work):

Open terminal/command prompt in this folder and run:
serve out -p 3000

========================================

💡 FEATURES:

✅ Complete project management dashboard
✅ Interactive Gantt chart with 2025 timeline
✅ Kanban boards for task management
✅ Team collaboration tools
✅ Focus and productivity tracking
✅ Modern, responsive design
✅ Works offline - no internet required

========================================

🆘 TROUBLESHOOTING:

Problem: "serve command not found"
Solution: Run "npm install -g serve"

Problem: Port 3000 in use
Solution: Use "serve out -p 8080" (different port)

Problem: Permission denied (Mac/Linux)
Solution: Run "chmod +x start-projectflow.sh"

========================================

📞 SYSTEM REQUIREMENTS:

- Node.js 18 or higher
- 2GB RAM minimum
- Modern web browser
- 500MB disk space
